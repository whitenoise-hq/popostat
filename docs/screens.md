# screens.md — 반려동물 전투력 측정기 (PopoStat · 포포스탯)

> 화면별 레이아웃·동작·디자인 규칙. `planning.md`(규칙)와 `tech-stack.md`(구현)를 화면 관점에서 연결.

---

## 0. 디자인 톤

- 카드 게임 느낌(포켓몬 카드). 등급별 색 차등, 상위 등급(S 이상) 화려한 연출.
- 전체 앱 톤은 가볍고 재밌게. 측정 결과 해설은 "능력 배틀물" 말투.
- 폰트/색은 Figma AI로 디자인 (소프트하되 카드 자체는 임팩트 있게).
- 스타일은 `StyleSheet.create` + `theme/colors.ts` 단일 출처. hex 직접 박지 않기. NativeWind 안 씀.
- 아이콘은 Ionicons. 카드 스탯·UI 어디에도 **이모지 쓰지 않기**(기기별 렌더링 불일치). 아래 와이어프레임의 ⚡·▓ 등은 표현용 표기일 뿐, 구현 시 Ionicons/커스텀 바로 대체.

## 2. 측정 탭 (measure)

핵심 화면. 단계형 UI.

### 상태 흐름
```
[빈 상태] → 사진 선택 → [사진 있음, 이름 입력 대기]
  → 이름 입력 완료 → [측정 가능] → 측정 버튼
  → [스캔 중] → [결과 카드]
```

### 레이아웃
- 상단: 타이틀 / 짧은 안내 ("우리 아이 전투력은?")
- 사진 영역:
  - 빈 상태: 큰 점선 박스 + "카메라" / "앨범" 두 버튼
  - 선택 후: 미리보기 썸네일 + 다시 선택 버튼
- 이름 입력: 텍스트 인풋 (placeholder "이름을 입력하세요")
- 하단 고정: **[전투력 측정]** 버튼
  - 사진+이름 모두 있어야 활성화, 아니면 비활성

### 스캔 연출 (중요)
- AI 응답 2~5초 → 대기를 재미로.
- 카드 뒷면 + 스캔 게이지/번쩍임 애니메이션 (reanimated 또는 Lottie)
- "전투력 측정 중..." 등 멘트 로테이션

### 결과
- 완성된 카드 등장(플립/등장 연출)
- 버튼: [덱에 저장됨 확인] / [공유하기] / [다시 측정하기(새 사진)]
- 동물 미검출 시: 카드 대신 친화적 에러 ("동물을 못 찾았어요. 다른 사진을 올려보세요")

## 3. 카드 디자인 (등급별)

### 공통 구조
```
┌─────────────────────────┐
│ [이름]            [등급] │
│ ┌─────────────────────┐ │
│ │      펫 사진         │ │
│ │            ⚡power   │ │
│ └─────────────────────┘ │
│ [칭호]            [품종] │
│ 공격 ▓▓▓▓▓░░ 78         │
│ 방어 ▓▓▓░░░░ 64         │
│ 민첩 ▓▓▓▓░░░ 71         │
│ 귀여움 ▓▓▓▓▓▓ 92        │
│ 게으름 ▓▓▓░░░ 55        │
│ [필살기명] — 설명        │
└─────────────────────────┘
```

### 등급별 색 (테두리/배경/강조)
| 등급 | 톤 |
|------|-----|
| F | 회색 |
| D | 청록 |
| C | 파랑 |
| B | 보라 |
| A | 코랄/주황 |
| S | 골드 + 광택 애니메이션 |
| SS | 핑크-퍼플 홀로그램 + 글로우 |
| SSS | 무지개 회전 홀로그램 + 강한 글로우 |

### 화려함 연출 (S 이상)
- 광택(shine) 스윕/회전 애니메이션
- 글로우(테두리 발광)
- 구현: `expo-linear-gradient` + `react-native-reanimated`, 필요 시 Lottie

## 4. 덱 탭 (deck)

수집한 카드 컬렉션.

### 레이아웃
- 카드 그리드 (2열) 또는 리스트 — 사용자 토글 가능(선택)
- 카드 탭 → 상세(큰 카드 + 공유 버튼)
- 정렬: 전투력순 / 등급순 / 최신순
- 필터: 등급별, 동물 종류별(강아지/고양이/기타)
- 빈 상태: "아직 카드가 없어요. 측정해서 첫 카드를 만들어보세요" + 측정 탭 이동 CTA

### 카드 상세
- 전체 카드 크게
- [공유하기] (이미지 캡처 → 공유)
- [삭제] (소유자만, 확인 후) — 삭제는 사용자 본인 행동으로만
