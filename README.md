# ⚡ 포포스탯 (PopoStat)

**우리 아이의 숨은 전투력을 측정하라**

반려동물 사진 한 장이면 AI가 스탯을 분석해 **전투력 카드**를 만들어줍니다.
포켓몬 카드처럼 등급이 매겨지고, 측정한 카드는 **덱**에 모아 수집할 수 있어요.

## 주요 기능

- **AI 전투력 측정** — 사진을 올리면 AI가 공격·방어·민첩·귀여움·게으름 5가지 스탯을 분석합니다
- **전투력 카드** — 포켓몬 카드 형태로 결과를 확인. 전투력에 따라 **F ~ SSS 8단계 등급**, 상위 등급일수록 화려한 연출
- **덱 수집** — 측정한 카드를 덱에 모으고 등급별로 정렬·필터
- **카드 공유** — 완성된 전투력 카드를 이미지로 저장하고 자랑하세요
- **능력 배틀물 해설** — 진지한 척하지만 웃긴 AI의 카드 해설과 필살기

> 공정성을 위해 AI는 0~100 스탯만 매기고, **전투력(power)과 등급(grade)은 코드로 계산**합니다. 같은 사진을 다시 측정해도 점수가 흔들리지 않아요.

## 기술 스택

- **프론트엔드**: React Native + Expo (SDK 56), TypeScript, Expo Router
- **상태 관리**: TanStack Query (React Query)
- **백엔드**: Supabase (Auth, PostgreSQL, Storage, Edge Functions)
- **인증**: 카카오 로그인 + Sign in with Apple
- **AI**: OpenAI GPT-4.1 Mini (사진 분석 → 스탯 산정, 전투력·등급은 코드 계산)
